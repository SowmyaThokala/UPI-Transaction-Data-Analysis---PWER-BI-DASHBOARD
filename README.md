Overview

This project presents a Power BI dashboard for analyzing UPI (Unified Payments Interface) Transactions data. The dashboard enables users to explore transaction patterns, amounts, and categories using interactive visuals and synced slicers across multiple pages. It provides a dynamic and user-friendly reporting experience through bookmarks and navigation features.

## Business Objective

The main goal of this dashboard is to provide insights into UPI transaction performance and user behavior. The report allows stakeholders to:

Analyze UPI transaction trends and categories.

Interactively filter and compare data using slicers.

Seamlessly navigate between visuals through bookmarks.

View synchronized data insights across multiple pages.

## Data Source and Setup

Imported the UPI Transactions CSV file into Power BI Desktop.

Cleaned and transformed the data within Power BI.

Established the necessary data model for analysis and visualization.

## Report Development in Power BI
Slicers Configuration

Added 10 slicers to the report for dynamic filtering.

Renamed and reshaped the slicers using Canvas Settings from the Format your report page option in the Visualizations pane.

Copied and pasted the slicers to Page 3 to maintain consistent filtering.

Used the Sync Slicers option under the View tab to synchronize slicers across all pages (Page 1, Page 2, and Page 3).

## Visualizations

Matrix Visual:
Added to display multiple columns organized into Rows, Columns, and Values, enabling multidimensional data analysis.

Line Chart (Page 1):
Created an initial line chart to represent transaction trends over time.

Column Chart:
Duplicated the line chart and modified it into a column chart for comparative analysis.

## Bookmarks and Navigation

Created four bookmarks using the Bookmarks option from the View tab to capture different visual states of the report.

Used the Selection pane to manage visibility of visuals by disabling other charts for each bookmark.

Updated respective charts in each bookmark to maintain consistent views.

Created a Bookmark Navigator using the Buttons option under the Insert tab for easy navigation between bookmarks.

## Key Features

Interactive slicers synchronized across multiple pages.

Bookmark-based navigation for dynamic report switching.

Matrix, Line, and Column charts for detailed analysis.

User-friendly and visually appealing dashboard design.
